# Installation

MassLooker requires [PHP](https://www.php.net/) 5.6 to run.

```sh
Follow These Steps for Installation
$pkg install php
$pkg install git
$pkg install mc
$ git clone https://github.com/sanjidtk/masslooker
$ cd masslooker
$ php login.php
$ php run.php
```
